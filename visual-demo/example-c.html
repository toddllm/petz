Example C: World Generation Scenario

Creating New Terrain

When a player explores uncharted territory, Luanti generates new MapBlocks using procedural algorithms. This process creates terrain, places ores, adds decorations, and saves everything to the database.

Trigger position: Player moves to (5000, 10, 5000) - virgin territory
Block position: (312, 0, 312)
Generation: New surface terrain with caves and ores

1

Block Not Found

When the cache and database lookups fail, Luanti triggers world generation for the missing block.

// src/servermap.cpp MapBlock *ServerMap::getBlockOrEmerge(v3s16 p) { MapBlock *block = getBlockNoCreateNoEx(p); if (!block) { // Block doesn't exist - must generate! m_emerge->enqueueBlockEmerge(PEER_ID_INEXISTENT, p, true); } return block; }
2

Initialize Map Generator

The EmergeThread wakes up and initializes the appropriate map generator (v7 in this case).

// src/emerge.cpp void *EmergeThread::run() { MapgenParams *mgparams = m_server->getEmergeManager()->mgparams; Mapgen *mapgen = createMapgen(mgparams->mg_name, mgparams, m_emerge); // mapgen is now an instance of MapgenV7 }

3

Generate Terrain with Perlin Noise

Multiple 3D Perlin noise functions create realistic terrain height maps and cave systems.

// src/mapgen/mapgen_v7.cpp void MapgenV7::generateTerrain() { // Generate base terrain height noise_terrain_base->perlinMap2D(node_min.X, node_min.Z); noise_terrain_alt->perlinMap2D(node_min.X, node_min.Z); for (s16 z = node_min.Z; z <= node_max.Z; z++) for (s16 x = node_min.X; x <= node_max.X; x++) { float terrain_height = baseTerrainLevelFromMap(index2d); for (s16 y = node_min.Y; y <= node_max.Y; y++) { if (y <= terrain_height) vm->m_data[i] = MapNode(c_stone); else if (y <= water_level) vm->m_data[i] = MapNode(c_water_source); else vm->m_data[i] = MapNode(CONTENT_AIR); } } }
4

Generate Caves

3D Perlin noise carves out cave systems through the solid terrain.

// src/mapgen/cavegen.cpp void CavesRandomWalk::makeCave() { float d1 = 6; // Cave width for (u16 j = 0; j < tunnel_routepoints; j++) { v3s16 cp(floor(vec.X + 0.5), floor(vec.Y + 0.5), floor(vec.Z + 0.5)); // Carve out cave carveRoute(vec, sqrt(d1) * route_y_min); // Random walk to next position vec += v3f(ps.range(-1, 1), ps.range(-1, 1), ps.range(-1, 1)); } }
5

Place Ore Deposits

Various ore types are scattered through stone using different distribution patterns.

// src/mapgen/mg_ore.cpp void OreScatter::generate(MMVManip *vm, int mapseed, u32 blockseed, v3s16 nmin, v3s16 nmax) { PcgRandom pr(blockseed + 4234); int volume = (nmax.X - nmin.X + 1) * (nmax.Y - nmin.Y + 1) * (nmax.Z - nmin.Z + 1); int csize = clust_size; int nthresh = clust_scarcity * volume / csize; for (int i = 0; i != nthresh; i++) { int x0 = pr.range(nmin.X, nmax.X - csize + 1); int y0 = pr.range(nmin.Y, nmax.Y - csize + 1); int z0 = pr.range(nmin.Z, nmax.Z - csize + 1); // Place ore cluster for (int z1 = 0; z1 != csize; z1++) for (int y1 = 0; y1 != csize; y1++) for (int x1 = 0; x1 != csize; x1++) { if (vm->m_data[i].getContent() == c_wherein) vm->m_data[i] = MapNode(c_ore); } } }
6

Apply Surface Decorations

Add grass, dirt layer, and decorations like trees and plants to the surface.

// Add dirt with grass on surface for (s16 z = node_min.Z; z <= node_max.Z; z++) for (s16 x = node_min.X; x <= node_max.X; x++) { int surface_y = findGroundLevel(v2s16(x, z)); if (surface_y > water_level) { vm->m_data[vi] = MapNode(c_dirt_with_grass); // Add 3 layers of dirt below grass for (int i = 1; i <= 3; i++) { vm->m_data[vi - i * ystride] = MapNode(c_dirt); } } }

Build NameIdMapping

Analyze all nodes in the block and create the optimal ID mapping for storage.

// src/mapblock.cpp void MapBlock::serialize(std::ostream &os, u8 version, bool disk, int compression_level) { // Build mapping of used nodes NameIdMapping nimap; if (disk) { for (u32 i = 0; i < nodecount; i++) { content_t c = data[i].getContent(); const std::string &name = nodedef->get(c).name; nimap.set(c, name); } // Optimize: reassign IDs starting from 0 data[i].setContent(nimap.getId(name)); } }

Compress and Save

Compress the block data with ZSTD and save to SQLite database.

// Compress with ZSTD std::ostringstream os_compressed(std::ios_base::binary); compressZstd(os_raw.str(), os_compressed, compression_level); // Save to database bool MapDatabaseSQLite3::saveBlock(const v3s16 &pos, const std::string &data) { s64 key = getBlockAsInteger(pos); sqlite3_bind_int64(m_stmt_write, 1, key); sqlite3_bind_blob(m_stmt_write, 2, data.c_str(), data.size(), NULL); return sqlite3_step(m_stmt_write) == SQLITE_DONE; } // Result: 16KB ‚Üí 1.2KB compressed // Saved with key: 5242905128000
